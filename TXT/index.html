Journey

2019

韓國出道、《The Dream Chapter: Star》

3月4日，官方Youtube頻道公開〈CROWN〉MV，在YouTube公開24小時內觀看次數突破1510萬。 在正式出道後，於社交平台Twitter上全球的流行趨勢前三名也是關於TOMORROW X TOGETHER，並且 登上全世界共44個國家及地區的iTunes排行榜1位。

10月21日以首張正規專輯《The Dream Chapter: MAGIC》回歸樂壇。

10月22日，《The Dream Chapter: MAGIC》登上全世界共25個地區的iTunes排行榜1位，成為韓國新人團體最高紀錄。

2020

日本出道、迷你二輯《The Dream Chapter: ETERNITY》、迷你三輯《minisode 1: Blue Hour》

1月15日，發行日本出道單曲《MAGIC HOUR》，這張作品空降 Oricon單日榜季軍位置，並登上日本朝日電視台的音樂直播節目《MUSIC STATION》演出。

2月15日，出道迷你專輯主打歌〈CROWN〉MV觀看次數突破一億，成為TOMORROW X TOGETHER首支破億MV。

5月19日，迷你二輯《The Dream Chapter: ETERNITY》登上全世界共51個地區的iTunes排行榜1位，刷新 了自身的最高紀錄。

10月26日，發行迷你三輯《minisode 1: Blue Hour》回歸樂壇，10月27日，主打曲〈Blue Hour〉進入MelOn 24Hits，使TOMORROW X TOGETHER成為第四位男團進入MelOn 24Hits的榜單。

2021

日語正規專輯《STILL DREAMING》、《The Chaos Chapter: FREEZE》

2月23日，日語專輯《STILL DREAMING》空降「Billboard 200」排行榜第173位，是TOMORROW X TOGETHER第 三張進入該榜單的專輯，亦是第二位以日語專輯進榜的韓國藝人。

5月26日，第三張迷你專輯主打歌〈Blue Hour〉MV觀看次數突破一億，成為TOMORROW X TOGETHER第二支破億MV。

6月14日，根據《告示牌》報導TOMORROW X TOGETHER正規二輯The Chaos Chapter: FREEZE於「Billboard200」 名列為第五名，成為出道3年以來於「Billboard 200」最高的排名專輯，亦是TOMORROW X TOGETHER第四張進榜的專輯。

2022

迷你四輯《minisode 2: Thursday's Child》、第一次世界巡演

2月6日，首張正規專輯主打歌〈Run Away〉MV觀看次數突破一億，成為TOMORROW X TOGETHER第三支破億MV。

4月26日，Big Hit娛樂於官方社群網站公告TOMORROW X TOGETHER將於7月2日起在首爾、芝加哥、紐約、亞特蘭大、 達拉斯、休斯敦、舊金山、洛杉磯展開首場世界巡迴演唱會《TOMORROW X TOGETHER WORLD TOUR 'ACT:LOVESICK'》。

5月9日，發行迷你四輯《minisode 2: Thursday's Child》回歸樂壇，主打歌為〈Good Boy Gone Bad〉。

8月28日，出道迷你專輯後續曲〈Cat & Dog〉MV觀看次數突破一億，成為TOMORROW X TOGETHER第四支破億MV。

2023

「The Name Chapter」系列、第二次世界巡演、

1月16日，Big Hit 娛樂於官方社群網站公告TOMORROW X TOGETHER將於3月25日起在首爾、新加坡、台北、日本 及美國的多個城市，進行第二次世界巡迴演唱會《TOMORROW X TOGETHER WORLD TOUR 'ACT:SWEET MIRAGE'》。

1月27日，發行迷你5輯《The Name Chapter: TEMPTATION》回歸樂壇，主打歌曲為〈Sugar Rush Ride〉，是時隔八個月的回歸。

3月6日，迷你5輯主打歌〈Sugar Rush Ride〉MV觀看次數突破一億，成為TOMORROW X TOGETHER 第五支破億MV。

9月17日，正規二輯主打歌〈0X1=LOVESONG〉MV觀看次數突破一億，成為TOMORROW X TOGETHER 第六支破億MV。

10月13日，發表正規三輯《The Name Chapter: FREEFALL》，並釋出主打歌Chasing That Feeling MV。

2024

迷你六輯《minisode 3: TOMORROW》、PPULBATU、第三次世界巡迴、迷你七輯《The Star Chapter: SANCTUARY》

4月1日，發行迷你六輯《minisode 3: TOMORROW》回歸樂壇，主打歌為〈Deja Vu〉，並於當晚舉行回歸Showcase。 專輯的所有收錄歌曲在發行當天進入Melon的百大單曲榜、主打歌《Deja Vu》排名第10。

4月29日，改版專輯《The Chaos Chapter: FIGHT OR ESCAPE》，主打曲〈LO$ER=LO♡ER〉MV觀看次數突破一億， 成為TOMORROW X TOGETHER第七支破億MV。

5月3日，TOMORROW X TOGETHER開啟在首爾、美國、日本、澳門、新加坡、印尼、台灣展開第三次世界巡迴演唱會《TOMORROW X TOGETHER WORLD TOUR 》。

8月1日，公布了TXT各成員的象徵性娃娃（PPULBATU）。

9月19日，然竣以《YEONJUN’s Mixtape: GGUM》出道，成為隊內首位Solo出道的成員。

11月4日，發行迷你七輯《The Star Chapter: SANCTUARY》回歸樂壇。

12月18日，《minisode 1: Blue Hour》的主打歌〈Blue Hour〉突破兩億，成為TOMORROW X TOGETHER首支觀看次數突破兩億的MV。

2025

團體休息、歐巡

1月5日，在結束第39屆金唱片獎後，全員正式進入長假休息。

1月12日，《The Name Chapter: FREEFALL》的先行單曲Back for More突破一億觀看，成為TOMORROW X TOGETHER第八支破億MV。

3月7日，在歷經1月5日開始的長假後，全員透過 - EP.2 仁川場重啟活動，並開始歐巡。
